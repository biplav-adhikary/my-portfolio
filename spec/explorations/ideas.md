# Ideas

Creative ideas worth considering. Not committed — just captured.

---

## Day/Night Cycle

The hero could shift its palette based on the visitor's local time:

- **Morning** (6am–12pm): Bright sky, fresh greens, warm light
- **Afternoon** (12pm–5pm): Full daylight, the default palette
- **Evening** (5pm–8pm): Sunset tones, amber sky, longer shadows
- **Night** (8pm–6am): Deep blues, muted greens, stars or fireflies instead of clouds

**Tradeoff:** Beautiful if done well. Risk of being distracting or making content harder to read in dark modes. Would need careful contrast testing.

---

## Seasonal Themes

Subtle shifts based on the calendar month:

- **Spring**: Cherry blossoms drifting, brighter greens
- **Summer**: Fuller landscape, warmer light, cicada-like ambient
- **Autumn**: Falling leaves, amber/rust palette shift
- **Winter**: Bare trees, muted colors, snow particles

**Tradeoff:** Delightful for repeat visitors. High maintenance. Could be done with CSS custom properties fed by a date check.

---

## Interactive Wind

Mouse movement creates a "wind" effect — clouds drift in the direction of cursor movement, grass elements sway.

**Tradeoff:** Feels magical when subtle. Feels like a gimmick when overdone. Would need to be very gentle.

---

## Sound Design

Optional ambient sound that the user can toggle:

- Soft wind, distant birds, gentle chimes
- A small speaker icon in the corner — off by default

**Tradeoff:** Controversial. Most users don't want sound. But the _option_ with good off-by-default behavior could be a memorable touch.

---

## Project Pages

Instead of showing everything on a single page, each project could be a mini-page with its own atmosphere:

- Unique hero illustration per project
- Deeper narrative with code snippets or diagrams
- A "return to trail" link back to main page

**Tradeoff:** Much more content to create and maintain. But would elevate the portfolio significantly.

---

## Easter Eggs

Small, hidden interactions for curious visitors:

- Clicking a specific cloud reveals a message
- A Konami code triggers a special animation
- A hidden page at `/studio` with development notes

**Tradeoff:** Fun. Time-consuming. Worth doing exactly one.

---

## Cursor Trail

A very subtle trail effect — not sparkles, but maybe tiny leaves or dots of light that fade quickly.

**Tradeoff:** Usually terrible. But with the right restraint (barely visible, only on desktop), could add to the atmosphere.
